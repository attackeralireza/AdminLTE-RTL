# نحوه استفاده از امکان راست چین 

در مسیر زیر کافیست فایل `custom.css` را به قسمت `head` اضافه کنید.

``dist\css\custom.css``

این قالب نسبت به قالب های راست چین دیگر به نسبت بهتر و به جزئیات توجه بیشتری شده است.
